# /media — official screenshots (editorial use)

Rules for this folder:
1. Source ONLY from Rockstar's official published galleries (Newswire / media page).
2. Filenames: keep Rockstar's original name or `gtavi-<subject>-NN.jpg`.
3. Attach to an article in `content/mock.ts` via:
   heroImage: { src: '/media/<file>.jpg', alt: '<what it shows>', credit: 'Rockstar Games' }
   The credit renders automatically on the article page.
4. Editorial context only. Never in the logo, icons, OG brand imagery,
   app-store creative, or anywhere brand-level — original art only there.
